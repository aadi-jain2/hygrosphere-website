# Image Setup Guide

## Where to Place Images

Place your team member photos in the `images` folder inside the `hygrosphere-website` directory:

```
hygrosphere-website/
├── index.html
├── styles.css
├── script.js
└── images/
    ├── avyana.jpg
    ├── ayanaa.jpg
    ├── ishaan.jpg
    ├── raghav.jpg
    └── samaira.jpg
```

## File Names (Case-Sensitive)

The images must be named exactly:
- `avyana.jpg` (or .png, .jpeg)
- `ayanaa.jpg`
- `ishaan.jpg`
- `raghav.jpg`
- `samaira.jpg`

## Supported Formats

- .jpg
- .jpeg
- .png

## Troubleshooting

1. **Check the file location**: Make sure images are in `hygrosphere-website/images/` folder
2. **Check file names**: Names must match exactly (case-sensitive)
3. **Check file extensions**: Use .jpg, .jpeg, or .png
4. **Refresh browser**: Press Ctrl+F5 to hard refresh
5. **Check browser console**: Press F12 and look for 404 errors

## If Images Still Don't Show

The website will automatically show placeholder boxes with names if images can't be found. If you see the placeholder boxes, the HTML/CSS is working correctly - you just need to add the image files.
