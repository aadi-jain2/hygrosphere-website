## Image Requirements

- **Format**: JPG or PNG
- **Recommended size**: 400x400 pixels or larger (square format works best)
- **Aspect ratio**: 1:1 (square) is preferred for the circular display
- **File naming**: Use lowercase names exactly as listed above

The images will be displayed in a circular frame on the team section. If an image is missing, a placeholder with the person's name will be shown automatically.
